# Intelcia Gradient Generator

Générateur de gradients basé sur le système chromatique Intelcia.

## Fonctionnalités

- **Système fermé** : Sélection préalable des couleurs autorisées parmi la palette Intelcia
- **Génération aléatoire** : Création de gradients uniquement à partir des couleurs sélectionnées
- **Accent distinct** : Option pour une couleur d'accent en début de gradient (style "i" d'Intelcia)
- **Configuration** : Angle, étapes d'interpolation, courbe d'easing
- **Logo adaptatif** : Le logo Intelcia s'affiche en blanc ou noir selon la luminosité du gradient
- **Mode background seul** : Visualisation du gradient sans le logo
- **Export CSS** : Code CSS copiable en un clic

## Palette Intelcia

| Couleur    | Hex       |
|------------|-----------|
| Jaune      | `#FFCC00` |
| Orange     | `#F5A623` |
| Rose       | `#E91E8C` |
| Violet     | `#9B59B6` |
| Bleu       | `#3498DB` |
| Turquoise  | `#1ABC9C` |
| Vert       | `#2ECC71` |

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Utilisation

1. Sélectionner les couleurs autorisées en cliquant dessus
2. Configurer l'angle, les étapes et l'interpolation
3. Cliquer sur "Generate" pour créer un gradient aléatoire
4. Copier le CSS généré
